Description:

Dynamic Web Application - We used two frameworks, including Vue.js for tables and illustration and Skeleton for css.

Simultaneously, we use Leaflet(1.5.1) and Nominatim APIs for the map, pop-ups and lat/lng > real address changing.

For the database, we use SQLite3 to save our data of St. Paul crimes.

This application generally can illustrate specific information (crime type, date, location, police grid etc.) of each crime that happened over 5 years in St. Paul, marking them on the map and what's more, checked boxes are supplied which can be used to select certain types and locations by users.
